# 🌸 Paarsh Matrimony

Paarsh Matrimony is a full-stack matrimony web application designed to help users find compatible life partners through profile creation, smart search, and matchmaking features.

🔗 **Live Demo:** https://paarsh-matrimony.onrender.com

---

## 📌 Project Overview

Paarsh Matrimony is a modern matchmaking platform where users can:

- Register and create detailed profiles
- Set partner preferences
- Search and filter profiles
- View matching users
- Send interest requests
- Manage their account securely

This project is built to simulate a real-world matrimony system similar to popular matrimonial platforms.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Secure Authentication
- Profile Creation & Editing
- Upload Profile Photo
- Search Profiles by Filters (Age, Location, Religion, etc.)
- View Detailed User Profiles
- Send & Receive Interest Requests

### 🛠 Admin Features
- Admin Login
- Manage Users
- Approve / Reject Profiles
- Monitor Platform Activity
